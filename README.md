# 🐍 贪吃蛇游戏

一个使用原生 HTML5 Canvas 和 JavaScript 编写的经典贪吃蛇游戏。

## ✨ 特性

- 🎮 流畅的游戏体验
- 🎨 精美的视觉效果
- 📱 响应式设计，支持键盘控制
- 🏆 分数系统，最高分本地保存
- ⏸️ 暂停功能
- 🎯 碰撞检测

## 🚀 预览

游戏已部署至 GitHub Pages：

**[点击预览游戏](https://1124992339.github.io/snake-game/)**

## 🎯 操作说明

| 按键 | 功能 |
|------|------|
| ↑ ↓ ← → | 控制蛇的移动方向 |
| W A S D | 替代方向键 |
| 空格键 | 暂停/继续游戏 |
| Enter | 重新开始游戏 |

## 🛠️ 技术栈

- HTML5 Canvas
- 原生 JavaScript
- CSS3 动画效果
